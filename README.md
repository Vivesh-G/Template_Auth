# Template_Auth
A simple template of a login page authentication that can be used for building projects

# Tech Stack
- React
- Vite
- FastAPI
- SQLAlchemy
- argon2

# Setup
- Backend: `cd backend && uv sync`
- Frontend: `cd frontend && bun install`

# Running the Application
- Backend: `cd backend && uv run uvicorn main:app --reload`
- Frontend: `cd frontend && bun run dev`
